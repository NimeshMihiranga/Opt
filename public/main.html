Quick Guide

Everything you need to integrate Mezuka OPT into your website.

🔗 Get Your API Key

  1. Click "Add Your Website" button
  2. Fill in your website name & URL
  3. Verify your WhatsApp number via OTP
  4. Your API key is auto-generated
  5. Copy the provided HTML snippet
  6. Paste it into your website
GET /api/requestotp/:phone

📡 API Endpoints

  • Request OTP: GET /api/requestotp/:phone
  • Verify OTP: GET /api/verifyotp/:phone/:otp
  • Refresh Token: POST /api/refresh
  • Check Status: GET /api/optstatus/:phone
  • Opt Out: GET /api/optout/:phone
  • Health Check: GET /api/health

⚙️ How OTP Works

  1. User enters their WhatsApp number
  2. Your site calls our API with their number
  3. We send a 6-digit OTP via WhatsApp
  4. User enters OTP — we verify it
  5. We return an Access + Refresh Token
  6. Use tokens for authenticated requests

🔐 Security Features

  • OTP expires in 5 minutes
  • Max 5 attempts per OTP
  • 60s cooldown between requests
  • Access tokens expire in 15 minutes
  • Refresh tokens last 30 days
  • All hashed with SHA-256
